# Computer Vision and Image Processing Fundamentals

## Overview

This project demonstrates fundamental computer vision and image
processing techniques using Python, OpenCV, NumPy, TensorFlow/Keras,
and MATLAB.

## Techniques Implemented

- Image Resizing
- Convolution Operations
- Sobel Edge Detection
- Prewitt Edge Detection
- Laplacian Sharpening
- Image Enhancement
- Gamma Correction
- Deep Learning Classification
- Webcam-Based Object Recognition
